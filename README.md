# ZH Script Studio v3.0 (beta)

Fresh document-centric UI. Runs **alongside** the stable v2 extension
(`com.zhmotions.scriptstudiov3`, menu "ZH Script Studio v3 (Beta)") for
side-by-side comparison.

- Bengali-first: Bijoy → Unicode, dual language, bundled Bengali fonts
- Caption style chips (Pop/Karaoke/Type-on/Slide/Fade)
- Tabbed dock: Titles / Captions / Templates, script always visible

See `RELEASE-NOTES-v3.0-DRAFT.md` + `DEPLOY-CHECKLIST-v3.0.md`. Not yet shipped.
