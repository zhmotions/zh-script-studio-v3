# ZH Script Studio

A free, ZH-branded Adobe CEP extension for **Premiere Pro** and **After Effects**.
View your local script files (`.docx`, `.txt`, `.srt`) and send selected text straight
to the timeline as **editable, animated titles**.

## Features
- Open `.docx`, `.txt`, `.srt` scripts (auto-reloads your last file)
- **＋ Title** — selected text → one editable animated title at the playhead
- **≣ Batch** — each selected line → its own sequential title
- **⚑ Marker** — selected text → timeline marker
- **⧉ Copy** — copy selection
- **🎬 Template** — use your own `.mogrt` animation; saved permanently as chips
- **🎨 Mark text** — colour / highlight selected script text
- **⚙ Display** — font, size, line spacing, colour swatches, zoom
- Works in Premiere Pro & After Effects 25.0+

## Install
1. Download `ZHScriptStudio-v2.0.zxp`.
2. Install with any ZXP installer (e.g. ZXP/UXP Installer) or via Adobe's command line.
3. Restart Premiere Pro / After Effects.
4. Open **Window → Extensions → ZH Script Studio**.

> Self-signed build — your installer may show a trust prompt. This is normal for free community extensions.
