# ZH Script Studio v4.0.0

Stable 4.0 — Bengali-first captions & titles for Premiere Pro + After Effects.

## Viral caption styles (Bengali-safe)
- **Hormozi** — word-by-word yellow reveal (Range Selector: Based-On-Words, Ease Low 100, fills-over-strokes)
- **Karaoke · Beasty · Spotlight · Letter Rise** — one whole word per cue (clean, no mid-letter paint, Bengali conjuncts intact)
- **Neon · Chrome · Pop · Clean** — whole-layer effects
- No text-animator selector ever splits Bengali glyphs

## Auto Subtitle (Chirp 2)
- Long videos finish **reliably** — 1 chunk per poll + progress saved after each chunk (gateway-kill safe)
- Cues time-sorted + out-point clamped → no overlapping captions
- Temp WAVs auto-deleted (no more Mac-cleaner "junk" flags)

## Templates — zero-config
- Drop a `.mogrt`/`.ffx` + a same-named image/video (or `*-preview.*`) → auto-listed with a **live animated preview**, no manifest
- Manifest is override-only → deleting a file removes it from the panel
- Active/hovered templates keep their video preview (CEF muted-autoplay enabled)

## Distribution
- Download is a **.zip** (holds the .zxp + install readme) so Mac cleaners / Gatekeeper stop deleting it
- License emails: correct per-app link (scriptstudio_autosub* matched), bundle keys list a button per app
